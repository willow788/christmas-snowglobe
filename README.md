# Christmas Snow Globe Simulator

An interactive, animated Christmas-themed snow globe built using Python and Matplotlib.
This project focuses on procedural animation, user interaction, and visual aesthetics to create a cozy festive desktop experience.

---

## Features

* Continuous animated snowfall
* Adjustable snowfall speed using a slider
* Snow accumulation at the ground
* Gently swaying multi-layer Christmas tree
* Twinkling and pulsing tree lights
* Star topper and festive greeting text
* Day and Night mode toggle
* Mouse interaction to shake the snow globe

---

## Tech Stack

* Python 3
* NumPy
* Matplotlib

No external graphics engines or machine learning libraries are used.

---

## How to Run

1. Install dependencies

pip install numpy matplotlib

2. Run the program

python christmas_snowglobe.py

---

## Controls

Snow Speed: Use the slider at the bottom of the window
Shake Snow Globe: Mouse click anywhere in the window
Toggle Day/Night Mode: Press D
Exit: Close the window
